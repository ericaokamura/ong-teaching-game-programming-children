Por meio de oficinas de capacitação, crianças e adolescentes poderão aprender as ferramentas e técnicas de desenvolvimento de jogos por meio de plataformas como Scratch e Roblox Studio.

Além das oficinas, o aluno poderá ter contato com diversas linguagens de programação (Python, JavaScript) e softwares para desenvolvimento de games (Unity, Unreal)

e poderá também participar de Hackathons uma vez por mês.

Alguns Dos Programas Oferecidos

Programa Descrição Carga Horária Estimada
Primeiros Passos Ensinar a utilizar sistemas básicos de computador como Microsoft 365 proporcionando conhecimento para utilizar computadores no dia-a-dia. 15 horas
Programação Entender os algoritmos de programação e a lógica de programação necessária para criar desde sistemas pequenos até os mais evoluidos. 40 horas
Desenvolvimento Web Nesta formação o participante irá desenvolver competências técnicas para a criação de aplicações web, utilizando conceitos e tecnologias atuais de mercado. 40 horas
Desenvolvimento de Games Descubra quais áreas estão envolvidas no desenvolvimento de um jogo digital e como você pode ingressar em cada uma delas. Aprenda os principais conceitos de uma das áreas mais aclamadas do desenvolvimento de jogos: o game design. 40 horas
